Clan Talk is a new type of email list server with features that make the user interface operate inside an email inbox. Clan Talk is simple enough for your grandmother but robust enough for a fanatical geek community. As you converse on Clan Talk your conversations are archived for future reference on the server. Scheduling and planning events is easy with calendar integration. Private conversations and custom lists of contacts are quick and easy.

  • Single email address
  • User Interface functionality in your inbox
  • Scale the number of users easily
  • Accessible command structure
  • Easily share events, photos, and documents
  • Simple API for extensibility
